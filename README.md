# NeoConnect

**NeoConnect** is a staff feedback and complaint management platform. It provides employees with a secure way to submit complaints, track them, vote in polls, and see analytics. Management and case managers can handle cases efficiently, ensuring transparency and accountability.

---

## Features

### Staff
- Submit complaints with category, department, location, severity, and description.
- Option to submit complaints anonymously.
- Track complaints using a unique tracking ID.
- Vote in polls created by Secretariat/Management.

### Secretariat / Management
- View all incoming cases in an inbox.
- Assign cases to Case Managers.
- Create and manage polls.
- Upload meeting minutes.

### Case Manager
- View assigned cases.
- Update case status, add notes, and close complaints.
- 7-day escalation for unaddressed cases.

### Admin (IT)
- Manage user accounts and security settings.

### Public Hub & Analytics
- View resolved cases and quarterly digest.
- Analytics dashboard showing cases per department, status, and category.
- Poll results displayed in charts.

---

## Tech Stack

- **Frontend:** Next.js, React, Chart.js  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **Deployment:** Vercel (Frontend), Render (Backend)
